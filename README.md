# Industrial Vision Inspection System

Industrial computer vision inspection system using OpenCV, classical ML, PyTorch basics, FastAPI inference API, and a simple inspection dashboard.

## Goal

Build a practical, employer-friendly computer vision project for industrial inspection and quality-control scenarios.

## Planned Features

- OpenCV image preprocessing
- Rule-based defect detection
- Feature extraction
- Classical ML baseline
- PyTorch basics
- Model evaluation
- Error analysis
- FastAPI inference service
- Inspection dashboard
